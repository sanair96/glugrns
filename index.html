What is GLUG?

GLUG stands for GNU/Linux User's Group. GLUG RNSIT is a Student Group @RNSIT monitored by FSMK (Free Software Movement Karnataka) spreading the idea of Free Software Philosophy.

What is Free Software?

Free software, freedom-respecting software, software libre, or libre software is computer software distributed under terms that allow users to run the software for any purpose as well as to study, change, and distribute the software and any adapted versions. The right to study and modify software entails access to its source code
